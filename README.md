# 🌍 World Happiness Index Analysis

## 📌 Overview
This project analyzes the World Happiness Index dataset to understand what factors contribute to happiness across different countries.

It uses exploratory data analysis (EDA) and visualization to uncover meaningful insights.

---

## 🎯 Objectives
- Analyze global happiness trends
- Identify factors influencing happiness
- Explore correlations between features
- Visualize country-wise happiness scores

---

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
